# 🏥 Patient Management System (Python)

## 📌 Project Overview
This is a simple **Patient Management System** developed using Python.  
It helps in managing patient records, scheduling appointments, and generating billing details.

The system uses **file handling** to store data permanently in a text file.

---

## 🚀 Features
- ➕ Add new patients  
- 📄 View all patient records  
- 🩺 Schedule doctor appointments  
- 💵 Generate hospital bills  
- 💾 Save data permanently using a text file  

---

## 🛠️ Technologies Used
- Python 3.x  
- File Handling  
- Dictionary Data Structure
